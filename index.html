The Box Model

Image courtesy of www.w3.org!

  1. Margin: This is the space around the element. The larger the morgin, the more spac between our element and the elements around it. The margin can be adjusted to move the HTML elements closer to or farther from each other.
  2. Border: This is the edge of the element. Ever time the border property is set, this is what is becoming visible.
  3. Padding: This is the space between the content and the border.
  4. Content: Things that are in the box. Example text of a paragraph.

TM stands for top margin, TB stands for top border, and TP stands for top badding. Top, Right, Left, Bottom. Padding, Border, and Margin.

Margins

When margins are adjusted, it moves the element relative to the other elements on the page and relative to the 'walls' of the HTML document. If an HTML element with a specific width and set its margin to auto, it tells teh document to automatically put equal left and right margins on our elemetn, centering it on the page.
